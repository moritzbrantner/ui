## Defaults

- Mobile breakpoint remains `768px`.
- Components should stack, wrap, or clamp on `360px` and `390px` viewports without requiring consumer props.
- Dense tables, matrices, maps, charts, and graph-like components may use explicit internal scroll containers.
- Overlay content should clamp to the viewport and scroll internally before it can create document-level overflow.
